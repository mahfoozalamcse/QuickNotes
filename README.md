# Notes Application

A professional Full-Stack Notes Management Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This project demonstrates real-world implementation of RESTful APIs, database integration, and scalable frontend architecture.

---

## Project Description

This application allows users to manage personal notes through a simple and responsive interface. Users can create, view, edit, and delete notes while experiencing secure and optimized performance using rate limiting and structured API design.

---

##  Key Features

- Full CRUD Operations for Notes  
- RESTful API Implementation  
- MongoDB NoSQL Database Integration  
- Rate Limiting using Upstash Redis  
- Responsive User Interface  
- Clean & Scalable Folder Structure  
- Error Handling & Status Code Implementation  
  

---

## Concepts Implemented

- MERN Stack Architecture  
- REST API Design Principles

## Installation & Setup

### 1. Clone The Repository
  git clone https://github.com/mahfoozalamcse/QuickNotes.git
  cd QuickNotes
  1. 
  cd backend
  npm install
  npm run dev

  2.
  cd frontend
  npm install
  npm start
### .env

  MONGO_URI=your_mongodb_connection_string
  UPSTASH_REDIS_URL=your_redis_url
  UPSTASH_REDIS_TOKEN=your_redis_token
  PORT=5000
